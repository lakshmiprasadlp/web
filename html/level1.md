# sup-sub 

```
<p>
        (a+b)2 = a2 + b2 + 2ab
    </p>
    <p>
        (a+b)<sup>2</sup> = a<sup>2</sup> + b<sup>2</sup> + 2ab
    </p>
    <p>
        CH4 + O2 = H20 + CO2
    </p>
    <p>
        CH<sub>4</sub> + O<sub>2</sub> = H<sub>2</sub>0 + CO<sub>2</sub>
    </p>
```
